# Sistema de Gestão de Tarefas em PHP

Aplicação web desenvolvida em PHP 8+ com foco em organização de código, boas práticas de engenharia de software e desenvolvimento backend.

## Funcionalidades
- CRUD de tarefas
- Marcação de tarefas como concluídas
- Filtro por status

## Tecnologias
- PHP 8+
- MySQL
- PDO
- HTML/CSS
- Git

## Objetivo
Projeto desenvolvido para estudo prático de PHP e arquitetura MVC,
reforçando fundamentos de backend e lógica de programação.
